# 참참참! (Neon)

이미지에 적힌 규칙 그대로 만든 **3레인 회피 게임**입니다.

## 실행

`charm/index.html`을 브라우저에서 열면 바로 실행됩니다.

## 조작

- 키보드: `← / ↓ / →` 또는 `A / S / D`
- 모바일: 아래 3개 버튼 터치

## 규칙

- 위에서 내려오는 오브젝트가 있는 라인과 **같은 라인**을 고르면 **OUT(라이프 -1)**
- **다른 라인**을 고르면 **라이프 +1** + 점수 증가
- 선택 제한시간 기본 **5초** (라운드가 올라갈수록 조금 빨라짐)
